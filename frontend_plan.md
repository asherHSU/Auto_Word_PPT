# 前端開發計畫

好的，由於前端目錄是空的，我將從頭開始建立前端應用程式。以下是我的前端開發計畫：

1.  **選擇框架與工具**: 我將使用 React (搭配 TypeScript) 框架，並使用 Vite 進行快速專案設置。
2.  **基本應用程式結構**: 建立一個基礎的 React 應用程式，包含路由設定。
3.  **身份驗證介面**: 開發用戶註冊和登入介面，與後端 API 進行互動。
4.  **歌曲管理介面**: 
    *   顯示歌曲列表 (從後端 `/api/songs` 獲取數據)。
    *   提供歌曲搜尋功能。
    *   提供新增、編輯和刪除歌曲的表單，與後端 `POST /api/songs`, `PUT /api/songs/:id`, `DELETE /api/songs/:id` 端點互動。
5.  **檔案生成介面**: 
    *   允許用戶選擇或輸入歌曲名稱，以生成文件。
    *   呼叫後端 `/api/generate` 端點並處理生成文件的下載。
6.  **基本樣式**: 使用簡潔的 CSS 樣式或輕量級 CSS 框架，確保 UI 乾淨且功能完善。
